# Projet Lightbox

## Description
Un mini-site statique développé en HTML, CSS et JavaScript présentant
une galerie interactive d'images de lampes LED. Développé en équipe
de 4 étudiants dans le cadre d'un projet Git/GitHub collaboratif.

## Installation
1. Ouvrir VS Code
2. Ouvrir le terminal dans VS Code (Terminal → New Terminal)
3. Sélectionner Git Bash comme terminal
4. Cloner le dépôt :
```
   git clone https://github.com/limitlessXe/lightbox-project.git
```
5. Ouvrir le dossier du projet dans VS Code :
```
   code lightbox-project
```
6. Ouvrir `index.html` dans votre navigateur pour voir le site.
