# Projet Lightbox

## Description
Un mini-site statique développé en HTML, CSS et JavaScript présentant
une galerie interactive d'images de lampes LED. Développé en équipe
de 4 étudiants dans le cadre d'un projet Git/GitHub collaboratif.
